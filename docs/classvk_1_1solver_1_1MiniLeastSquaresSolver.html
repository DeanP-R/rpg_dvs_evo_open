template<int D, typename T, typename Implementation>
class vk::solver::MiniLeastSquaresSolver< D, T, Implementation >

Abstract Class for solving nonlinear least-squares (NLLS) problems. Template Parameters: D : dimension of the residual, T: type of the model e.g. SE2, SE3
