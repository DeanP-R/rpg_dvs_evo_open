size_t svo::DepthFilterOptions::max_search_level = 2

Specify the max pyramid level for the matcher. Normally, you don't need to change this parameters.

◆ seed_convergence_sigma2_thresh

double svo::DepthFilterOptions::seed_convergence_sigma2_thresh = 200.0

Threshold for the uncertainty of the seed. If seed's sigma2 is thresh smaller than the inital sigma, it is considered as converged. Default value is 200. If seeds should converge quicker, set it to 50 or if you want very precise 3d points, set it higher.

◆ use_inverse_depth

bool svo::DepthFilterOptions::use_inverse_depth = true

Use inverse-depth parametrization for seeds. Default is true. Set to false if you are using the depth-filter to reconstruct small objects with fixed depth range.
