Detailed Description

Settings for the IMU statistics Check the following references for more information about the IMU parameters:

  • https://github.com/ethz-asl/kalibr/wiki/IMU-Noise-Model-and-Intrinsics
  • http://www-users.cs.umn.edu/~trawny/Publications/Quaternions_3D.pdf (Sec. 2.1) Default parameters are for ADIS16448 IMU.
