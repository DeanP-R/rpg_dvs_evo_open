void svo::SparseImgAlignBase::setMaxNumFeaturesToAlign ( int  num)
inline

Set number of features used for the alignment. If a number is <= 0 all features are used Specifying a number gives you the control about the performance of the alignment. Having an alignment running over 600 or 100 features doesn't make that much a difference
